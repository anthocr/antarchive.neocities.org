Resources

Below are some resources that I've used in my site or I found useful. Many Neocities sites also have a variety of resources that they host themselves for others to use for free. However, it's important to not hotlink other peoples assets, since this can slow down people's sites due to the server having to fetch the resources. Please just be wary of this when using others' graphics.

Coding

Petra Pixel's Coding Tutorials I personally used the tutorial on using a GitHub repo to deploy to Neocities, this allows you to easily use any IDE to code your webiste (I use VSCode)

Graphics

Broider A free pixel border resource with hundreds of user generated borders to choose from. I used the .magic border on my homepage.

Tools

Sreenfly A tool to test your website's responsiveness on different screen sizes.
